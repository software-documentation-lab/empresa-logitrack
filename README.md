# empresa-logitrack

## Setor
Logística e Transporte

## Contexto do problema
A LogiTrack opera múltiplos sistemas para roteirização, monitoramento de frota, gestão de entregas e atendimento ao cliente. A ausência de documentação arquitetural dificulta onboarding de novos profissionais, análise de impacto e tomada de decisão técnica.

## Objetivo do desafio
Produzir documentação de arquitetura que descreva componentes, integrações, fluxos críticos e restrições técnicas do cenário atual.

## Instruções resumidas para os grupos
1. Faça um fork deste repositório para a conta do grupo.
2. Trabalhe exclusivamente no fork do grupo.
3. Preencha a documentação na pasta `docs/`, usando `docs/architecture-template.md` como referência e consolidando o resultado em `docs/architecture.md`.
4. Entregue o link do fork do grupo conforme orientação da disciplina.
