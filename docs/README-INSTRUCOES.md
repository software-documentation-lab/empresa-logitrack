# Instruções da Atividade

Este repositório faz parte da atividade prática da disciplina de **Documentação Técnica em DevOps**.

## Objetivo do grupo
Documentar a arquitetura da LogiTrack de forma clara e auditável, permitindo entendimento de estrutura, dependências e riscos técnicos.

## Documentação que deve ser produzida
- Arquivo final: `docs/architecture.md`
- Referência de estrutura: `docs/architecture-template.md`

## Regras da simulação
- Esta atividade é uma simulação acadêmica.
- Não é necessário desenvolver código ou implementar soluções.
- O foco está em documentação de arquitetura e comunicação técnica.

## Forma de trabalho
- O trabalho deve ser realizado no fork do grupo.
- O grupo deve registrar premissas e lacunas quando faltar informação no cenário.
